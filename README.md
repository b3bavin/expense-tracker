# Expense Tracker

A simple Python Tkinter app to track monthly expenses.

## Features
- Add and delete expenses easily
- Predefined categories: Food, Travel, Study, Entertainment, Others
- Set a monthly spending limit
- Shows remaining balance (green if safe, red if exceeded)
- Total monthly expenses displayed
- Automatic reset at the start of each new month
- Category-wise pie chart
